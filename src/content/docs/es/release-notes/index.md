Esta página contiene los registros de iteración e historial de versiones de la plataforma SenseCraft HMI.

## v1.0 - 2025-10-24

**Diseño y lista de páginas rediseñados para una creación más fácil**

- Nuevo diseño de UI para una navegación más fácil
- Agregar función de lista de páginas, crear múltiples UIs como diapositivas de keynote
- Agregar bibliotecas de páginas, guardar todas tus UIs

**Editor de UI mejorado e integración de datos**

- Soporte para diseños de canvas tanto horizontales como verticales
- Soporte para importación y exportación de plantillas de UI
- Extraer directamente datos en vivo de SenseCAP desde tu cuenta
- Agregar hora de última conexión del dispositivo y hora de siguiente actualización

**Soporte de hardware ampliado**

- Soporte para ePaper DIY Kit-EE04, compatible con pantallas de 7.3'' y 7.5''
- Soporte para TRMNL 7.5'' (OG) DIY Kit con pantalla de 7.5''

**Página de herramienta de flasheo de firmware y procesamiento de imágenes**

- Soporta flasheo de firmware de SenseCraft HMI y firmware TRMNL
- Agregar herramienta de difuminado para procesar imágenes y exportar directamente a archivos x.c

## v0.6

### Lanzamiento inicial
- **Soporte de dispositivos**: reTerminal E1001 y E1002
- **Tipos de página**: Soporte individual para galería, canvas, RSS y página web
- **Gestión de firmware**: Flasheo y actualizaciones de firmware en línea
